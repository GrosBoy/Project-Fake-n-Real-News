#Итоговый проект: Определение фейковых новостей
##Описание
Разработать автоматизированную систему или алгоритм, способный с высокой точностью отличать достоверную информацию от преднамеренно ложной, вводящей в заблуждение или непроверенной (фейковой) в текстовом контенте, маскирующемся под новости.

##Данные
Датасет "Fake and Real News dataset" из сайта Kaggle

##Цель
Предсказать классификацию новостей как правда или ложь,

##Используемые технологии
pandas, numpy, sklearn, matplotlib, seaborn, keras, tensorflow, nltk, LSTM, LogisticRegression, TfidfVectorizer

##Pipeline проекта
1. Загрузка и подготовка данных:
- Загрузка датасета "Fake and Real News" с Kaggle
- Объединение данных о фейковых и правдивых новостей
- Добавление меток (0 - правда, 1 - фейк)
- Случайная выборка 5000 записей для ускорения работы
2. Предобработка текста:
- Приведение к нижнему регистру
- Удаление всех символов, кроме букв и пробелов
- Токенизация с помощью nltk.word_tokenize
- Лемматизация (приведение слов к базовой форме)
- Удаление стоп-слов (английских)
3. Векторизация текста:
- Bag-of-Words (CountVectorizer) - создание матрицы частот слов
- TF-IDF (TfidfVectorizer) - создание матрицы с учетом важности слов
4. Подготовка данных для нейронных сетей:
- Токенизация текста с ограничением в 10000 слов
- Добавление паддинга (дополнение нулями до длины 40 токенов)
- Разделение данных на обучающую и тестовую выборки (80/20)
5. Построение и обучение модели:
- Создание модели LSTM с архитектурой:
-- Embedding слой (10000 слов → 64 измерения)
-- LSTM слой (64 нейрона)
-- Dropout (30%)
-- Dense слой (32 нейрона, ReLU)
-- Dropout (30%)
-- Выходной слой (1 нейрон, sigmoid)
- Компиляция с бинарной кросс-энтропией и оптимизатором Adam
- Обучение в течение 3 эпох с batch_size=128
6. Оценка результатов:
- Визуализация точности на обучающей и валидационной выборках
- Анализ распределения предсказанных вероятностей
- Классификационные метрики (точность, полнота, F1-score)

##Результаты
Результат не особо хороший получился, потому что больше половины фейковых новостей неправильно угадывает, судя по графику Confusion Matrix, но это можно исправить в дальнейшем.
